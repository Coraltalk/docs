## Latest Release

### Version 2.0.0 - May 2026

## Release Notes Format

Each release includes:

- **✨ New Features** - Major new functionality
- **🔧 Improvements** - Enhancements to existing features
- **🐛 Bug Fixes** - Resolved issues and bugs
- **⚠️ Breaking Changes** - Changes requiring action (when applicable)
